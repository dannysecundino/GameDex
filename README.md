# 🎮 GameDex — Sistema de Avaliação de Jogos

## 📌 Descrição

O **GameDex** é um sistema desenvolvido para catalogar e gerenciar avaliações de jogos eletrônicos, permitindo o registro, consulta e organização de informações sobre diferentes títulos.

O projeto foi desenvolvido com foco na aplicação de conceitos de **Programação Orientada a Objetos (POO)**, como encapsulamento, modularização e organização de responsabilidades.

---

## 🎯 Objetivo

* Aplicar conceitos de Programação Orientada a Objetos
* Desenvolver um sistema organizado e escalável
* Trabalhar com modelagem de entidades do mundo real (jogos e avaliações)

---

## 🚀 Funcionalidades

* ✔️ Cadastro de jogos
* ✔️ Cadastro e manutenção de dados de usuários durante a execução
* ✔️ Registro de avaliações
* ✔️ Consulta de informações dos jogos
* ✔️ Organização dos dados de forma estruturada
* ✔️ Execução simples via classe principal

---

## 🛠️ Tecnologias Utilizadas

* Linguagem: **Java** *(ajusta se não for Java)*
* Paradigma: Programação Orientada a Objetos
* Execução via terminal/IDE

---

## ⚙️ Como Executar

### 🔹 Pré-requisitos

* Java instalado *(ou a linguagem correta do projeto)*
* IDE (IntelliJ, Eclipse) ou terminal

---

### 🔹 Execução

1. Clone o repositório:

```bash
git clone <link-do-repositorio>
```

2. Acesse a pasta do projeto:

```bash
cd GameDex
```

3. Execute a classe principal:

```bash
run Main
```

Ou execute diretamente pela sua IDE rodando a classe `Main`.
